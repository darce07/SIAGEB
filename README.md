# Sistema de Monitoreo

Proyecto Frontend en React + Vite + TailwindCSS para gestionar formularios de monitoreo con guardado local.

## Requisitos
- Node.js 18+
- npm

## Instalación
```bash
npm install
```

## Ejecutar en desarrollo
```bash
npm run dev
```

## Rutas disponibles
- `/monitoreo`: pantalla de selección de monitoreo.
- `/monitoreo/ficha-escritura`: formulario principal de monitoreo.

## Notas
- El estado del formulario se guarda en `localStorage` para uso offline/local.
- La estructura de estado está lista para integrarse con un backend/API posteriormente.
